# Methodology

This folder contains the core documents for the informativeness classification framework.

Files:

- `informativeness-classification-criteria.md`: canonical editable version for the project pipeline and AI agents (English).
- `informativeness-classification-criteria-pt.md`: Portuguese reference version for team discussion.
- `manual-annotation-protocol.md`: manual annotation sampling strategy and execution workflow.

Usage guideline:

- Prefer editing and versioning the English Markdown file;
- Keep the Portuguese Markdown file as human reference.
